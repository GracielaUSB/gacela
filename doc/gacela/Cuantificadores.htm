Los cuantificadores (E)#

Los cuantificadores permiten escribir de manera compacta expresiones sobre la aplicación de un operador asociativo, conmutativo y con elemento neutro a un rango de valores.

Eqnt x1,...,xn : R1,...,Rm : E}}} x1,...xn variables,
R1,..,Rm,E expresiones y
qnt un cuantificador que es un identificador
precedido del símbolo "%"

Cuantificadores predefinidos (qnt)#

qnt%sigma operador +, neutro 0ejemplo
qnt%pi operador *, neutro 1ejemplo
qnt%forall operador /\, neutro trueejemplo
qnt%exists operador \/, neutro falseEjemplo
qnt%max operador max, neutro MIN_INT
qnt%max operador max, neutro MIN_DOUBLE
qnt%min operador min, neutro MAX_INT
qnt%min operador min, neutro MAX_DOUBLE
qnt%unionoperador union, neutro {}
qnt%sigma operador +, neutro 0.0
qnt%sigma operador +, neutro ""
qnt%pi operador *, neutro 0.0

Los cuantificadores pueden ser escritos con letras mayúsculas o minúsculas.

Con la incorporación del analizador de tipos en la versión v1.1, los cuantificadores %Dsigma, %Dpi, %Dmax, %Dmin y %cat desaparecen.

Abreviaciones#

E(%count x1,...,xn : R1,...,Rm : E) == (%sigma x1,...,xn : R1,...,Rm : (E=>1|0))
E(%first x : R : E) == (%min x : R : (E=>x|MIN_INT))ejemplo
E(%last x : R : E) == (%max x : R : (E=>x|MAX_INT))ejemplo

USB-GCL v1.1

E(%set x1,...,xn : R1,...,Rm : E) == (%union x1,...,xn : R1,...,Rm : {E})

Propiedades de los cuantificadores#

Sea Op un operador asociativo que recibe un par de argumentos de tipo t y produce un valor de tipo t.
Sea Nop un elemento neutro para Op (Es decir, para cualquier valor v de tipo t: v Op Nop = Nop Op v = v).

La operación Op es susceptible de utilizarse como cuantificador con la notación (%Op i : R : E) donde i es una variable que toma como valores todos aquellos que satisfacen el rango R y E es una expresión que determina los valores en donde Op opera.

Las siguientes propiedades son satisfechas por los cuantificadores:

  • Extracción 
 [R(i:=E1) ==> (%Op i : R : E) = E(i:=E1) Op (%Op i : R, i!=E1 : E)]
  • Rango Vacío 
 [(%Op i : false : E) = Nop]
