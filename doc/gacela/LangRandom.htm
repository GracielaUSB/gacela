La acción random#

La acción random se utiliza para generar aleatoriamente valores mayores o iguales a 0.0 y menores que 1.0 a una o más variables de tipo double.

Generación aleatoria random x1,...,xnx1...xn variables

Ejemplo#

 var d:=0.0 : double
;random d
;if (d>=0.5) -> write("Ganaste")
  | (d<0.5) -> write("Perdiste")
fi

Ejemplo#

    write("Mi turno de tirar los dados:")
    ;var r1:=0.0, r2:=0.0 : double
    ;random r1, r2
    ;var d1 := toInt(r1*6.0)+1, d2 := toInt(r2*6.0)+1 : int
    ;write((d1=d2) => "doble "+d1 | d1+" y "+d2)
