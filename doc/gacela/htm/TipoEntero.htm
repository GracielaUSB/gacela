Los enteros#

El tipo int sirve para representar enteros positivos y negativos dentro de un rango[1] dado.

Las constantes enteras son secuencias de dígitos ("[0-9]+" en notación de expresión regular).

[#1]Las constantes predefinidas MIN_INT y MAX_INT representan el menor entero y el mayor entero representable en GCL respectivamente.

La aritmética en el tipo int es circular: MAX_INT+1=MIN_INT.

Los operadores disponibles sobre los enteros son:

OperadorEjemplodef(operador)
- int {x=5} y:= -x {y=-5} def(-x)==def(x)
int ^ int {x=5 /\ y=2} x,y:= x^y,y^x {x=25 /\ y=32} def(a^b)==(b>0 \/ (b=0 /\ a!=0))
int * int {x=5 /\ y=2} x:= y*x {x=10} def(x*y)==def(x)/\def(y)
int div int {x=5 /\ y=6} x,y:= x div 2,y div 2 {x=2 /\ y=3} def(x div y)==y!=0
int mod int {x=5 /\ y=6} x,y:= x mod 2,y mod 2 {x=1 /\ y=0} def(x mod y)==y!=0
int max int {x=5 /\ y=6} x:= x max y {x=6} def(x max y)==true
int min int {x=5 /\ y=6} x:= x min y {x=5} def(x min y)==true
abs(int) {x=5 /\ y=-5} x,y:= abs(x),abs(y) {x=5 /\ y=5} def(abs(x))==true
int + int {x=5 /\ y=-2} x:= y+x {x=3} def(x+y)==true
int - int {x=5 /\ y=-2} x,y:= x-y,y-x {x=7 /\ y=-7} def(x-y)==true
boolean => int | int {x=5 /\ y=-2} z := (x=y => x+1 | y+1) {z=-1} def(x=>y|z)==true
readInt(String) {true} y:= readInt("valor de y:") {MIN_INT<=y<=MAX_INT} def(resdInt(x))==true
toInt(double) {y=25.25} i:= toInt(y) {i=25} def(toInt(x))==true
intOfString(String) {s="50"} i:=intOfString(s) {i=50} def(intOfString(x))==true

Los cuantificadores siguientes producen valores de tipo int:

CuantificadorEjemplo
(%sigma i : R : int) {x=(%sigma i: 0<=i<k : i)} x:= x+k {x=(%sigma i: 0<=i<k+1 : i)}
(%pi i : R : int) {x=(%pi i: 0<=i<k : i)} x:= x*k {x=(%pi i: 0<=i<k+1 : i)}
(%max i : R : int) {x=(%max i: k<=i<N : a[i]) /\ k>0 } x:= x max a[k-1] {x=(%max i: k-1<=i<N : a[i])}
(%min i : R : int) {x=(%min i: k<=i<N : a[i]) /\ k>0 } x:= x min a[k-1] {x=(%min i: k-1<=i<N : a[i])}
(%count i : R : int) {(%count i: 0<=i<10 : i mod 3 = 1)=3}
(%first i : R : int) (%first i: 0<=i<10 : i*i>i+i)=3

Los predicados siguientes se utilizan para comparar enteros:

PredicadoEjemplodef(predicado)
int < int x<MAX_INT-1 ==> x < x+1 def(x<y)==true
int > int x>MIN_INT+1 ==> x > x-1 def(x>y)==true
int <= int x<=y == x = y \/ x < y def(x<=y)==true
int >= int x>=y == x = y \/ x > y def(x>=y)==true
int != int x!=x+1 def(x!=y)==true
int = int MIN_INT<=(x*y)<=MAX_INT ==> x = (x*y)/y def(x==y)==true

El procedimiento write se utiliza para escribir enteros.

write( int ) { x=25 } write(x) escribe "25"
